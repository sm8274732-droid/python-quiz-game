# 🎯 Python Quiz MCQ Game

Ek interactive Python quiz game jo Python programming knowledge test karta hai!

## 🌟 Features

- **4 Categories**: Python Basics, Control Flow, Functions & OOP, Data Structures
- **20 Questions** total with detailed explanations
- **Timed Mode** - 15 seconds per question
- **High Score System** - Top 10 scores save hote hain (JSON file mein)
- **Grading System** - A+ se F tak grades
- **Random Mode** - Sab categories se mix questions
- **Colorful UI** - Terminal mein colored output

## 🚀 Kaise Chalayein

```bash
python quiz_game.py
```

## 📋 Requirements

- Python 3.6 ya usse upar
- Koi extra library install nahi karni!

## 🎮 How to Play

1. Program chalayein
2. Apna naam likhein
3. Category chunein
4. Kitne questions chahiye batayein
5. Timed mode on/off karein
6. Questions ka jawab A/B/C/D mein dein
7. Result aur grade dekho!
